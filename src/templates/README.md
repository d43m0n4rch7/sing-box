# sing-box Configuration File Descriptions

### Community Templates
* **`1.12-community-template.json`** — Template with FakeIP routing, adblocking, and geo-routing.
* **`1.13-community-template.json`** — Comprehensive baseline sing-box template.
* **`1.14-community-template.json`** — Advanced template with split-routing configuration.

### Baseline TUN & Dual-Stack Setup
* **`official-client-tun-ipv4.json`** — Minimal baseline configuration with an IPv4 TUN interface.
* **`official-client-tun-ipv4-ipv6.json`** — Configuration with dual-stack (IPv4/IPv6) support for TUN.
* **`official-client-tun-fakeip.json`** — Dual-stack FakeIP resolution engine (IPv4/IPv6).

### Advanced Routing & Bypass
* **`official-client-bypass.json`** — Integration of remote SRS rule-sets and split DNS routing.
* **`official-client-bypass-1.json`** — Localized fallback traffic routing via Google with ECS optimization.
* **`official-client-bypass-2.json`** — Expanded DNS hijacking, QUIC/STUN blocking, and direct routing configuration for CN traffic.
