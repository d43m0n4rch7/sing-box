# sing-box Routing Rules Descriptions

### Android Packages
* **`android-packages.json`** — Package rules for international Android applications.
* **`google-android-packages.json`** — Package rules for Google system services and Android AI applications.

### Domain Rules
* **`anthropic.json`** — Domain rules for Anthropic and Claude AI services.
* **`ext.json`** — Domain rules for external international services and general AI platforms.
* **`github.json`** — Domain rules for the GitHub development platform.
* **`google-ai.json`** — Domain rules for Google AI and Gemini platforms.
* **`google.json`** — Domain rules for core Google services and cloud infrastructure.
* **`jetbrains.json`** — Domain rules for JetBrains software and ecosystem.
* **`telegram.json`** — Domain rules for Telegram messenger.
* **`youtube.json`** — Domain rules for YouTube and its media delivery CDNs.
